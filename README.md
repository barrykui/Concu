# Concu
Concu   : Concurrent to run optionally numbers of commamdline.


``` shell

$ java -jar concu.jar   

Concu   : Concurrent to run optionally numbers of commamdline.
Version : 1.0.1
Usage   : 
 -f  <str>    Commandline list file.
 -t  <int>    Number of concurrent threads [1, 50]; 
 -w  <str>    Daemon name to be watched. eg.'java','wget'.
 -v           View output.
```
